# 🎨 QR Styling Studio - Clone & Customization
 
## 📝 Descrição do Projeto
Este projeto é uma recriação funcional e personalizada do **QR Styling Studio**, desenvolvido para explorar o potencial de customização avançada de QR Codes. O objetivo central foi replicar uma interface de estúdio profissional, permitindo que o usuário manipule não apenas as cores, mas a geometria, os "olhos" do código e a inclusão de logotipos centrais.
 
Desenvolvido como um desafio de **Desenvolvimento Front-end Avançado**, o sistema lida com a renderização dinâmica em Canvas/SVG, garantindo que as modificações estéticas não quebrem a integridade dos dados contidos no QR Code. O foco foi proporcionar uma ferramenta de branding poderosa para empresas que desejam fugir do padrão estático e monocromático.
 
 
## 🚀 Tecnologias Utilizadas
* **Linguagens:** HTML5, CSS3, JavaScript (ES6+)
* **Bibliotecas:** Styling-QRCode-JS (ou similar utilizada)
* **Ferramentas:** Git, VS Code
* **Conceitos:** Manipulação de Canvas, Exportação de Arquivos (PNG/SVG), State Management
 
## 📊 Resultados e Aprendizados
A construção deste clone permitiu um aprofundamento técnico em como os dados são codificados visualmente.
* **Manipulação de Geometria:** Implementei a troca de estilos dos módulos (quadrados, arredondados, pontos) de forma dinâmica.
* **Gerenciamento de Imagens:** Aprendi a lidar com o upload e posicionamento de logos sobre a matriz do QR Code com cálculo de margem de segurança.
* **UX de Dashboard:** Foquei na organização de menus colapsáveis para manter a interface limpa enquanto oferece dezenas de opções de customização.
 
 
## 🔧 Como Executar
1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Para build de produção:
   ```bash
   npm run build
   ```
