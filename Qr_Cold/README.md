# 📱 QR Cold - Gerador de QR Codes com Acessibilidade
 
## 📝 Descrição do Projeto
O **QR Cold** é uma aplicação focada na geração dinâmica de QR Codes, desenvolvida com o objetivo de unir funcionalidade técnica e identidade visual. Diferente de geradores genéricos, este projeto prioriza a **acessibilidade** e a integração de **brand kits**, permitindo que marcas e utilizadores criem códigos que respeitem a sua paleta de cores e logotipia sem comprometer a escaneabilidade.
 
Desenvolvido como um projeto estratégico de portfólio, o sistema foca na experiência do utilizador (UX), garantindo que a geração de códigos para links, textos ou dados de contacto seja intuitiva e rápida. A lógica do software foi estruturada para suportar diferentes níveis de correção de erro, garantindo que o código funcione mesmo em condições adversas de impressão ou brilho.
 
 
## 🚀 Tecnologias Utilizadas
* **Linguagens:** HTML5, CSS3, JavaScript (ES6+)
* **Bibliotecas:** QRCode.js / QR-Code-Generator
* **Ferramentas:** VS Code, Git/GitHub
* **Conceitos:** Design Responsivo, UI/UX, Manipulação de DOM
 
## 📊 Resultados e Aprendizados
O projeto representou um marco no domínio de manipulação de ativos visuais via código e lógica de front-end.
* **Customização de Brand Kit:** Implementei a lógica necessária para que o utilizador altere cores de fundo e do código em tempo real.
* **Foco em Acessibilidade:** O sistema foi pensado para gerar códigos com alto contraste, facilitando a leitura por diferentes dispositivos e aplicações.
* **Otimização de Performance:** A geração do QR Code ocorre de forma instantânea no lado do cliente (Client-side), sem necessidade de requisições pesadas ao servidor.
 
 
## 🔧 Como Executar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Para build de produção:
   ```bash
   npm run build
   ```
