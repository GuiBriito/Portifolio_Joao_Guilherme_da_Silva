# 📦 Sistema de Gestão de Estoque & Exportação CSV
 
## 📝 Descrição do Projeto
Este projeto é um sistema de inventário robusto desenvolvido em **C**, projetado para gerenciar o fluxo de mercadorias de forma eficiente. Ele permite o cadastro, alteração, exclusão e busca de produtos, utilizando arquivos de texto para persistência de dados, o que garante que as informações não sejam perdidas após o encerramento do programa.
 
Desenvolvido como um projeto de **Estrutura de Dados e Manipulação de Arquivos**, o sistema implementa algoritmos de ordenação (*Bubble Sort*) para organizar o estoque por diferentes critérios (nome ou preço) e uma funcionalidade estratégica de exportação para **CSV**. Isso permite que os dados gerados no software sejam facilmente integrados a ferramentas de análise como Microsoft Excel ou Google Sheets.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** C (Padrão C99)
* **Bibliotecas:** `stdio.h`, `stdlib.h`, `string.h`, `ctype.h`
* **Ferramentas:** Compilador GCC, Git
* **Conceitos:** Manipulação de Arquivos (File I/O), Estruturas de Dados (Structs), Algoritmos de Ordenação.
 
## 📊 Resultados e Aprendizados
A construção deste sistema permitiu o domínio de conceitos fundamentais de engenharia de software e persistência de dados.
* **Manipulação de Arquivos:** Implementei a leitura e escrita em arquivos `.txt` utilizando delimitadores, permitindo a gestão de um banco de dados local simples.
* **Persistência e ID Único:** Desenvolvi uma lógica para geração automática de IDs incrementais, garantindo a integridade referencial de cada produto cadastrado.
* **Algoritmos de Busca e Ordenação:** Apliquei buscas *case-insensitive* e múltiplas variações do algoritmo *Bubble Sort* para otimizar a visualização dos dados pelo usuário.
* **Interoperabilidade:** A função de exportação CSV foi um marco, transformando dados brutos de um programa console em um formato de mercado amplamente aceito.
